Programming

What is programming ?

A programming language is a formal language comprising a set of instructions that produce various kinds of output. Programming languages are used in computer programming to implement algorithms.

  • Software Devloper

    A software developer plays a key role in identifying, designing, installing and testing a software system that has been built for a company from the ground up.

  • Web Devloper

    Web developers have a similar job to software developers; however they specialise in creating websites.

  • Security Devloper

    Security developers specialise in creating systems, methods and procedures to test the security of software systems and fix security flaws.

Languages We Learn about

C

C programming is a general-purpose, procedural, imperative computer programming language developed in 1972 by Dennis M. Ritchie at the Bell Telephone Laboratories to develop the UNIX operating system.

C++

C++ is a middle-level programming language developed by Bjarne Stroustrup starting in 1979 at Bell Labs.

Java

Java 8 is the most awaited and is a major feature release of Java programming language. This is an introductory tutorial that explains the basic-to-advanced features of Java 8 and their usage in a simple and intuitive way.

Python

Python is a general-purpose interpreted, interactive, object-oriented, and high-level programming language. It was created by Guido van Rossum during 1985- 1990.
